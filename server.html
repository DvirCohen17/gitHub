How to Set Up the Server

Setting up the CCE server is simple and straightforward. Just follow these steps to get your collaborative coding environment up and running in no time:

1. Download and Extract: First, click the "Download Server" button above to download the server zip file. Once the download is complete, extract the contents of the zip file to a location on your computer where you want to store the server files. This can be any folder of your choice, but make sure it's a location you can easily access.

2. Run the Server: Inside the extracted folder, you will find a file named server.exe. Simply double-click this file to run the server. A console window should appear, indicating that the server is up and running. You don't need to install anythingâ€”just run this executable file.

3. Check Your IP Address: To allow others to connect to your server, you'll need to provide them with your IP address. To find it, open the Command Prompt (you can do this by searching for "cmd" in the Start menu). In the Command Prompt, type ipconfig and press Enter. Look for the "IPv4 Address" under your active network connection. This is the IP address you'll share with your friends so they can join your server.

4. Start Collaborating: Now that your server is running and you have your IP address, you're ready to start coding together with your friends! Simply share the IP address with them, and they'll be able to connect to your server using the CCE app.

And that's it! You've successfully set up the CCE server and are now ready to enjoy seamless collaborative coding with your team.

If you encounter any issues or need further assistance, feel free to consult the CCE documentation or reach out to our support team.
